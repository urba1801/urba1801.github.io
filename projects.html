Projects
Hidden Water

The purpose of this project was to develop and deploy dataloggers that monitor water level in the sumps around the University of St.Thomas campus. My part of this project was programming the dataloggers to use sonar sensors in order to record the water levels of the sumps and send the data to monitormywatershed in order to see the data in real time. You can currently see the water levels for the sumps in OSS, Herrick Hall, and the library.

Global Treks & Adventures Travel Guide

The purpose of this project was to create and publish a travel guide for Iceland in order to suggest hiking routes and help with species identification. My part of this project was researching the common habitats and ecosystems of Iceland.
